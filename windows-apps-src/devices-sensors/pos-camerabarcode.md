# <a name="camera-barcode-scanner"></a>相機條碼掃描器
相機條碼掃描器是動態建立，只要 Windows 使用軟體解碼器配對連接到您電腦的相機。  每個相機 - 解碼器配對是完全可用的條碼掃描器。

|------|------------|
| [系統需求](pos-camerabarcode-system-requirements.md)  | Windows 版本清單，支援相機條碼掃描器，以及成功讀取條碼的相機需求。 |
| [快速入門](pos-camerabarcode-get-started.md)              | 相機條碼掃描器的逐步介紹 |
| [託管預覽](pos-camerabarcode-hosting-preview.md)          | 了解如何在您的應用程式中主控預覽相機條碼掃描器 |
| [啟用或停用](pos-camerabarcode-enable-disable.md)         | 了解如何啟用或停用隨附於 Windows 10 的預設軟體解碼器 |
| [支援的條碼符號表示](pos-camerabarcode-symbologies.md) | 本主題提供 Windows 10 隨附的軟體條碼解碼器所支援之每個條碼符號表示的範例條碼, 包括:UPC/EAN, 程式碼 39, 程式碼 128, 交錯2之 5, 資料條形全方向, 資料條形堆疊, QR 代碼和 GS1DWCode。 |
|
