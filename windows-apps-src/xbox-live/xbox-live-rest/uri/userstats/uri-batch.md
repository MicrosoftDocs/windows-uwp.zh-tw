# <a name="batch"></a>/batch
跨多個項目中存取多個播放程式統計資料的複雜的批次的要求。 這些 Uri 的網域是`userstats.xboxlive.com`。
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>有效的方法

[POST](uri-batchpost.md)

&nbsp;&nbsp;POST 跨多個項目可做為多個播放程式統計資料的複雜的批次要求的 GET 方法的方法。
