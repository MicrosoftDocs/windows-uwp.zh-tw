# <a name="lists-uris"></a>清單 URI
 
本節提供從 Xbox Live 服務的通用資源識別元 (URI) 位址和相關聯的超文字傳輸通訊協定 (HTTP) 方法的詳細*Pin*。
 
只有遊戲及 Xbox 360 上、 Windows Phone 裝置上、 Smartclass，或在 Xbox.com 上執行的應用程式可以使用這項服務。
 
這些 Uri 的網域是 eplists.xboxlive.com。
 
<a id="ID4EPB"></a>

 
## <a name="in-this-section"></a>本節內容

[/users/xuid(xuid)/lists/PINS/{listname}](uri-usersxuidlistspinslistname.md)

&nbsp;&nbsp;在清單中存取項目。

[/users/xuid(xuid)/lists/PINS/{listname}/ContainsItems](uri-usersxuidlistspinslistnamecontainsitems.md)

&nbsp;&nbsp;判斷一組項目 （項目識別碼所指定） 是否會包含在清單中而未擷取完整的清單。

[/users/xuid(xuid)/lists/PINS/{listname}/index({index})?insertIndex={insertIndex}](uri-usersxuidlistspinslistnameindex.md)

&nbsp;&nbsp;移動清單中的項目。

[/users/xuid(xuid)/lists/PINS/{listname}/RemoveItems](uri-usersxuidlistspinslistnameremoveitems.md)

&nbsp;&nbsp;從清單移除項目。
