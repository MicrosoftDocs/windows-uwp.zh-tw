# <a name="usersxuidxuiddeleteuserdata"></a>/users/xuid({xuid})/deleteuserdata
完全重設的測試使用者的信譽資料。 僅供測試。 這些 Uri 的網域是`reputation.xboxlive.com`。 連接埠 10443 一律會呼叫此 URI。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| xuid| 64 位元不帶正負號的整數| Xbox 使用者識別碼 (XUID) 正在刪除其資料的使用者。|
