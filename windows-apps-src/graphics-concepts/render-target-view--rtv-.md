# <a name="render-target-view-rtv"></a>轉譯目標檢視 (RTV)


轉譯目標讓場景轉譯到暫時中繼緩衝區，而不是轉譯到背景緩衝區，以轉譯到場景。 這個功能可讓您使用複雜的場景，可能轉譯為反映紋理或做為圖形管線內的其他用途使用，或可能是轉譯之前新增其他像素著色器效果至場景。
