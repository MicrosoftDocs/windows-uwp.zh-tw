# <a name="trustedplatformusersxuidxuidscidsscid"></a>/trustedplatform/users/xuid({xuid})/scids/{scid}
擷取此儲存體類型的配額資訊。 這些 Uri 的網域是`titlestorage.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| xuid| 不帶正負號的 64 位元整數| Xbox 使用者識別碼 (XUID) 播放程式的人員提出要求。| 
| scid| guid| 要查閱的服務組態的識別碼。| 
  
<a id="ID4E3B"></a>

 
## <a name="valid-methods"></a>有效的方法

[GET](uri-trustedplatformusersxuidscidsscid-get.md)

&nbsp;&nbsp;擷取此儲存體類型的配額資訊。
