協助工具
===============================================================  
\[ 針對 Windows 10 上的 UWP app 更新。 如需 Windows 8.x 文章，請參閱[封存](http://go.microsoft.com/fwlink/p/?linkid=619132) \]

介紹與通用 Windows 平台 (UWP) app 相關的協助工具概念。

協助工具可以針對無法或不便使用傳統使用者介面的殘障人士，讓他們也能使用應用程式。 根據法律的規定，某些情況必須提供協助工具， 不過無論法律的規定為何，最好能解決無障礙問題，這樣應用程式便能吸引更多的使用者。 此外，也會提供與您 app 協助工具相關的 Windows 市集宣告。

**注意** 將 app 宣告為無障礙 app，只與 Windows 市集有關。
