# <a name="getting-started-with-uwp-for-ios-developers"></a>開始使用適用於 iOS 開發人員的 UWP


如果您是考慮適用於 Windows 10 開發 iOS 開發人員，這些文件是絕佳起點。 它們會為您介紹一些需要知道當您開始撰寫應用程式的概念，他們將討論您如何發行您的工作，在 Microsoft Store。

在本節中，我們將漸進式開始並檢查如何使用 Microsoft Visual Studio 和 C# 程式設計語言來建立簡單的 app，特別是您目前使用之工具的程序可能會有哪些不同。 (不喜歡 C#？ 別擔心，其他程式設計語言和工具可供使用，我們將介紹在[開始使用：選擇程式設計語言](getting-started-choosing-a-programming-language.md))。

Windows 10 導入了新的平台，建立更吸引人的應用程式在桌上型電腦、 膝上型電腦、 平板電腦、 電話裝置等等。 由於通用 Windows 平台 (UWP) 應用程式提供許多獨特的功能，因此單純移植您的 iOS 應用程式將會失去這些功能。 建議您嘗試新的控制項和功能，看看它們如何讓您的開發工作更加容易，以及如何開發新的 app。

最重要的是：不要只是移植您的 app，而是**重新構想**您的 app，並充分利用這些新的功能和裝置。 不能勉強接受最小公分母，但建立豐富的體驗，透過 Cortana 使用唯一的 Windows 10 功能，例如動態磚、 通知和互動。

若要開始使用這些逐步解說，您必須具有 Windows 10 和 Microsoft Visual Studio 安裝的電腦。 您可以下載從[程式設計 UWP 應用程式的開發人員下載](https://developer.microsoft.com/en-us/windows/downloads)。 沒有 Windows 電腦嗎？ 不用擔心，您可以使用您的 Mac：請參閱[在 Mac 上安裝 Windows 和開發工具](setting-up-your-mac-with-windows-10.md)。
