# <a name="usersmescidsscidclips"></a>/users/me/scids/{scid}/clips

## <a name="uri-parameters"></a>URI 參數

| scid| 字串| 服務組態正在存取的資源識別碼。 必須符合 SCID 的已驗證的使用者。|
