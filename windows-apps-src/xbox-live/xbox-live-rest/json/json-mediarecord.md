## <a name="mediarecord"></a>MediaRecord
 
MediaRecord 物件具有下列規格。
 
| 成員| 類型| 描述| 
| --- | --- | --- | 
| id| 字串| Bing 目錄或提供者類別目錄所使用的媒體識別碼。| 
| idType| 字串| 如何解譯的媒體識別碼。 可能值包括 「 bing 」 和 「 提供者 」。| 
| name| 字串| 媒體內容的當地語系化的名稱。|
