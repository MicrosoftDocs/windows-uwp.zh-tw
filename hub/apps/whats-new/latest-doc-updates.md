# <a name="latest-updates-to-the-windows-developer-docs"></a>Windows 開發人員文件的最新更新

Windows 開發人員文件會定期更新，並提供新的和改進的資訊及內容。 以下是 2020 年 10 月 5 日的變更摘要。

注意：如需新增為 Windows 10 組建 19041 (也稱為 2004) 一部分的特定 API 清單，請參閱[此清單](/windows/uwp/whats-new/windows-10-build-19041-api-diff)。


## <a name="news"></a>新聞


本月我們將另一批絕對連結轉譯成相對連結，以便提升瀏覽速度並改善離線閱讀體驗。 如果您找到中斷的連結，請繼續讓我們知道：您可以直接從大部分的頁面提供意見反應，也可以透過我們控制的 [@WindowsDocs](https://www.twitter.com/windowsdocs) 在 Twitter 上聯繫我們。

* 已更新的文件：

    * [Surface Duo 文件](https://docs.microsoft.com/dual-screen/introduction)更新

    * [組合視覺效果](/windows/uwp/composition/composition-visual-tree)

    * [DirectML 常見問題集](/windows/win32/direct3d12/gpu-faq)
