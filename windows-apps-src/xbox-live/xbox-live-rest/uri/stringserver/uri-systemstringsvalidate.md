# <a name="systemstringsvalidate"></a>/system/strings/validate
存取驗證的字串陣列。 這些 Uri 的網域是`client-strings.xboxlive.com`。
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>有效的方法

[POST （/系統/字串/驗證）](uri-systemstringsvalidatepost.md)

&nbsp;&nbsp;可接受用於驗證的字串陣列，並傳回一個陣列的相同大小的結果。
