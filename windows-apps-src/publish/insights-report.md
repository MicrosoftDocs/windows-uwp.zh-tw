# <a name="insights-report"></a>見解報告


[合作夥伴中心](https://partner.microsoft.com/dashboard)中的 **深入** 解析報表會醒目顯示在您的收購、健康情況和/或使用量資料的過去30天內偵測到的特定度量 (增加或減少) 的重大變更。 這可讓您快速查看可能的重要變更，而不需要在這些報表中查看所有圖表。

> [!NOTE]
> 這份報表中的資料涵蓋過去30天。 您無法為此報表選取不同的時間週期。

報表會將資料分成三個索引卷 **標： [** 取得]、[ **健康** 情況] 和 [ **使用量** ] 若要查看其中一個區域的見解，請選取其索引標籤。

當偵測到資料的重大變更時，會顯示見解。 針對每個深入解析，我們將示範下列各項：
- **深入解析類型** ：偵測到見解的區域。
- **值** ：如果變更適用于整個 **深入解析類型** ) ，則會大幅變更 (或 **全部** 變更的特定度量。
- **日期** ：我們識別變更的日期。 此日期代表一周的結束時間，在此時間內，相較于那一周，我們偵測到大幅增加或減少。
- **整體影響** ：此值會在整個客戶群中增加或減少的百分比。 這可協助您瞭解特定變更的影響程度，尤其是在比較 **前幾名參與者** 所顯示的百分比資訊時。
- **前幾名參與者** ：適用的特定區段、封裝或其他識別因素，以協助您瞭解變更相關的客戶。 例如，可能會偵測到某個變更主要是來自特定市場或特定裝置類型的客戶。 針對 **健康** 情況資料，這可能包含特定的失敗雜湊或封裝版本。 在適用的情況下，我們也會顯示此係數的值增加或減少的百分比。
- **動作** ：
   - 選取 [ **顯示14天的趨勢** ] 以查看圖表，其中顯示計量在整個14天內如何變更，而導致深入解析日期。
   - 選取 [ **告訴我們] 是否正確** ，以提供您的意見反應，並讓我們知道我們提供的深入解析是否看似正確。 這種意見反應可協助我們持續改善我們在這裡提供的資料。
