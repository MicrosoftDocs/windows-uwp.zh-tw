# <a name="specular-light-maps"></a>反射光源對應


透過光源照明時，使用高度反射材質的光亮物件收到反射強光。 有時您可以透過套用反射光線對應至基本類型來取得更準確的強光，而不是使用由照明模組製造的反射強光。

若要執行反射光線對應，將反射光線對應新增至基本類型的紋理，然後調變（將結果乘以）RGB 光線對應。
