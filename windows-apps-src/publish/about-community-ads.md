# <a name="about-community-ads"></a>關於社群廣告

如果您的應用程式[會顯示橫幅插入式廣告](../monetize/display-ads-in-your-app.md)，您可以跨-升級您的應用程式與其他 Microsoft Store 中的應用程式的開發人員免費。 我們將此功能稱為「社群廣告」。  

以下是此計劃的運作方式：

* 在您選擇後到社群廣告，如下所述，您可以[建立免費的社群廣告活動](create-an-ad-campaign-for-your-app.md)。 您的應用程式接著會將分享促銷 ad 空間其他開發人員也選擇社群廣告。 您的應用程式將顯示參與社群廣告之其他開發人員所發佈的應用程式的廣告，而他們的應用程式也將會顯示您應用程式的廣告。
* 您可以透過在您的應用程式中顯示社群廣告來賺取其他應用程式中之推銷廣告空間的點數。 點數會依據以下程序計算：
  * 對於可使用提供社群廣告之應用程式的每個國家與地區，國家或地區的目前市場利率 eCPM (每千個曝光數有效成本) 值會乘以您的應用程式在該國家或地區的社群廣告要求數。 此值就是您為您的應用程式在該國家或地區賺取的點數。
  * 您在指定期間賺取的總點數等於您提供社群廣告的每個應用程式在每個國家或地區賺取之所有點數的總和。
* 您的點數會平均分配給所有正在進行的社群廣告行銷活動，並且會依據您社群廣告行銷活動所針對之國家/地區的目前市場利率 eCPM 值，轉換成您應用程式的廣告曝光數。
* 若要追蹤您 App 中社群廣告的績效，請參閱[廣告效益報告](advertising-performance-report.md)。

### <a name="opt-in-to-community-ads"></a>選擇加入社群廣告

您可以建立您的應用程式的其中一個社群 ad 行銷活動之前，您必須選擇上**Monetize** &gt; **應用程式內廣告**頁面[合作夥伴中心](https://partner.microsoft.com/dashboard)。

若要參加社群廣告對 UWP 應用程式：

1. 選取您的應用程式中使用，並向下捲動至 ad 單元**中繼設定**。
2. 如果**讓 Microsoft 將我的設定最佳化**已選取，社群廣告會為您的 ad 單元自動啟用。 否則，請選取基準組態或市場特定的組態**目標**下拉式清單，然後再檢查**Microsoft Community 廣告**方塊中**其他廣告網路**清單。

    > [!NOTE]
    > 您可以使用**權數**欄位，即可指定您想要顯示來自付費的網路與其他廣告網路，包括社群廣告的廣告的比率。

完成選擇後，您不需要重新發行應用程式。 一旦您已選擇加入，您可以在[建立廣告活動](create-an-ad-campaign-for-your-app.md)時選取**\[社群廣告 (免費)\]** 做為行銷活動類型。
