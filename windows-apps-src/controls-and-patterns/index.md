# 適用於 UWP app 的控制項和模式

在 UWP app 開發中，「控制項」<i></i>是顯示內容或啟用互動的 UI 元素。 控制項是使用者介面的基本要素。 我們提供超過 45 種控制項供您使用，從簡單的按鈕到強大的資料控制項 (例如資料格檢視) 都有。 「模式」<i></i>是可結合數個控制項以創造新項目的配方。

本節中的文章提供對您的 UWP app 新增控制項和模式的設計指導方針與程式碼撰寫指示。

以 XAML 和 C# 新增控制項並設定其樣式的一般指示。
