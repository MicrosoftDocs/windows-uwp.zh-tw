# <a name="usersowneridpeople"></a>/users/{ownerId}/people
存取呼叫者的使用者集合。 這些 Uri 的網域是`social.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| ownerId| 字串| 使用者正在存取的資源識別碼。 必須符合已驗證的使用者。 可能的值為"me"、 xuid({xuid}) 或 gt({gamertag})。| 
  
<a id="ID4EOB"></a>

 
## <a name="valid-methods"></a>有效的方法

[GET](uri-usersowneridpeopleget.md)

&nbsp;&nbsp;取得呼叫者的使用者集合。
