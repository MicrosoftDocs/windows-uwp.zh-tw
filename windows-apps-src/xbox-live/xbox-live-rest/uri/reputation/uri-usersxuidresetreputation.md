# <a name="usersxuidxuidresetreputation"></a>/users/xuid({xuid})/resetreputation
可讓強制執行小組，來存取指定的使用者的信譽分數。 這些 Uri 的網域和連接埠號碼是`reputation.xboxlive.com:10433`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| xuid| 字串| Xbox 使用者識別碼 (XUID) 指定的使用者。| 
  
<a id="ID4EVB"></a>

 
## <a name="valid-methods"></a>有效的方法

[POST (/users/xuid({xuid})/resetreputation)](uri-usersxuidresetreputationpost.md)

&nbsp;&nbsp;可讓強制執行小組，設定指定的使用者的信譽分數之後 （舉例來說） 帳戶攔截某些任意的值。
