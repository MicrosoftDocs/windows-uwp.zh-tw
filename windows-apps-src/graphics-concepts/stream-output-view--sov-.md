# <a name="stream-output-view-sov"></a>資料流輸出檢視 (SOV)


資料流輸出檢視可讓頂點、鑲嵌和幾何著色器已產生的頂點資訊串流輸出回到應用程式以供進一步使用。 例如已被這些著色器扭曲變形的物件可以寫回到應用程式，以提供更精確輸入給物理或其他引擎。 不過實際上，資料流輸出檢視是圖形管線不常使用的功能。
