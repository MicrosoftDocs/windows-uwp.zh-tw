# <a name="lighting"></a>光源


光線可用於照亮場景中的物件。 每個物件頂點的色彩是以目前紋理圖、頂點色彩及光線來源為基礎。

**注意：** 本節只是固定函式管線。 可程式著色器明確執行所有照明。
