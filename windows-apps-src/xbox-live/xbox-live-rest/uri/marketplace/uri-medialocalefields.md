# <a name="mediamarketplaceidfields"></a>/media/{marketplaceId}/fields
存取欄位的語彙基元。 這些 Uri 的網域是`eds.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| marketplaceId| 字串| 必要。 字串值取自<b>Windows.Xbox.ApplicationModel.Store.Configuration.MarketplaceId</b>。| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>有效的方法

[GET (/media/{marketplaceId}/fields)](uri-medialocalefieldsget.md)

&nbsp;&nbsp;取得欄位的語彙基元。
