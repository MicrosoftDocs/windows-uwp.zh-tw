# <a name="compute-shader-cs-stage"></a>計算著色器 (CS) 階段


計算著色器 (CS) 提供高速一般用途運算，並且利用了圖形處理器 (GPU) 大量平行處理器的優勢。 計算著色器階段提供了記憶體共用以及執行緒同步功能，允許更有效率的平行程式設計方法。

計算著色器可在許多執行緒上平行執行。

計算著色器是 [HLSL](https://msdn.microsoft.com/library/windows/desktop/bb509561)可程式化著色器階段，其使用 [DirectCompute](https://go.microsoft.com/fwlink/p/?linkid=209544) 技術將 Direct3D 的運用擴展超越圖形程式設計。
