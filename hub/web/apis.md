# <a name="apis-for-web-development"></a>Web 開發的 Api

這份清單涵蓋 Microsoft 內部可能對 網頁程式開發人員感興趣的各種 Api。 在 [Windows 10 的網頁開發總覽頁面](/overview.md)上尋找更多以 網頁程式開發為焦點的工具和服務。

- **[Academic Knowledge API](https://azure.microsoft.com/services/cognitive-services/academic-knowledge/)**<br>
使用學術知識 API，利用 Microsoft 學術圖形中豐富的學術內容。

- **[調適型卡片](https://adaptivecards.io/)**<br>
一種開放式卡片交換格式，可讓開發人員以通用且一致的方式交換 UI 內容，讓可操作的內容可以直接插入您的應用程式。

- **[ASP.NET Web Api](https://dotnet.microsoft.com/apps/aspnet/apis)**<br>
使用 c # 在任何平臺上建立安全的 REST Api。

- **[Azure DevOps Services REST API](/rest/api/azure/devops/)**<br>
Azure DevOps 可讓您透過內部部署和雲端中的原始檔控制、工作追蹤，以及持續整合和傳遞，共同作業軟體發展。

- **[Azure 媒體播放器](https://azure.microsoft.com/services/media-services/media-player/)**<br>
方便使用的 web 媒體播放機，可自動選擇最佳播放格式，包括 Flash 和 Silverlight 的回復，以及提供整合式內容保護。

- **[Bing Ads API](/advertising/guides/)**<br>
Bing Ads 是按按按 (PPC) 廣告平臺，用來根據使用者的搜尋查詢中所使用的關鍵字來顯示廣告。

- **[Bing 知識 Widget](https://www.bing.com/widget/knowledge)**<br>
利用 Bing 知識存放庫的強大功能來增強您的網站，並為您的使用者提供在頁面內容中偵測到的實體相關資訊。

- **[Bing 地圖服務 REST 服務 API](/bingmaps/rest-services/index.md)**<br>
用來執行工作的服務，例如地理編碼、反向地理編碼、路由和靜態影像。

- **[Bing Maps Web 控制項](https://www.bing.com/api/maps/sdkrelease/mapcontrol/isdk#overview)**<br>
專為商業智慧案例優化的 Microsoft 新式 web 地圖開發人員套件。

- **[Bing 搜尋 API](https://www.microsoft.com/bing/apis)**<br>
Bing 搜尋 Api 包括 web 搜尋、影像搜尋、新聞搜尋、自訂搜尋、自動建議、拼寫檢查、視覺搜尋、實體搜尋和影片搜尋。

- **[Bing 空間資料服務 REST API](/bingmaps/spatial-data-services/)**<br>
Batch 地理編碼、興趣點 (POI) 資料，以及儲存和公開空間資料的能力。

- **[認知服務 API](/azure/cognitive-services/)**<br>
在應用程式、網站和 bot 中建立智慧型和支援的演算法，以查看、聽取、說出、瞭解和解讀您的使用者需求。 涵蓋：視覺、語言、語音、決策、搜尋、認知服務容器，以及適用于大型資料的認知服務。

- **[電腦視覺 API](/azure/cognitive-services/computer-vision/)**<br>
讓開發人員能夠存取先進的演算法，以處理影像及傳回信息。

- **[Content Moderator API](/azure/cognitive-services/content-moderator/overview.md)**<br>
AI 提供的內容仲裁服務可掃描文字、影像和影片，並自動套用內容旗標，以及審核工具，也就是人類審核團隊團隊的線上仲裁者環境。

- **[臉部偵測 API](/azure/cognitive-services/face/)**<br>
偵測、辨識和分析影像中的人臉。

- **[知識探索服務](https://github.com/microsoft/cognitive-research-technologies-docs/blob/master/knowledge-exploration-service/GettingStarted.md)**<br>
啟用透過自然語言輸入對結構化資料進行互動式搜尋的功能。

- **[語言理解智慧型服務 (LUIS)](/azure/cognitive-services/luis/)**<br>
讓您的應用程式瞭解人在自己的單字中的意義。

- **[LinkedIn：新增至設定檔](https://addtoprofile.linkedin.com/)**<br>
按一下即可取得成員設定檔的認證和學位。

- **[Microsoft Graft API](/graph/use-the-api)**<br>
Microsoft Graph 是一個 RESTful Web API，可讓您存取 Microsoft Cloud 服務資源。

- **[摩納哥編者](https://microsoft.github.io/monaco-editor/)**<br>
摩納哥編輯器是支援 VS Code 和 F12 的程式碼編輯器。 包含語法顏色標示、豐富 IntelliSense 和差異編輯器的支援。

- **[Microsoft 商家中心內容 API](/advertising/shopping-content/)**<br>
內容 API 是一種 RESTful API，可讓廣告軟體以程式設計方式管理其 Microsoft 商家中心目錄。

- **[Microsoft 小組 API](/graph/teams-concept-overview)**<br>
使用 Microsoft Graph 與以聊天為基礎的 Microsoft 小組工作區整合。 建立虛擬小組來解決商務問題、新增適當的人員、設定通道、索引標籤和應用程式，以及在問題解決時封存或刪除團隊。

- **[個人化工具 Api](/azure/cognitive-services/personalizer/)**<br>
讓您的應用程式選擇最佳體驗來向使用者顯示，並從其即時行為中學習。

- **[Power BI Embedded](https://azure.microsoft.com/en-us/services/power-bi-embedded/)**<br>
運用 Power BI Embedded，您可以將 Power BI 報告整合至您的 Web 應用程式或行動應用程式。

- **[QnA Maker](/azure/cognitive-services/QnAMaker/)**<br>
幾分鐘內即可從半結構化內容到問題和解答 bot。

- **[Skype Web Api](/skype-sdk/websdk/docs/skypewebsdk)**<br>
一組 JavaScript Api 和 HTML 控制項，可建立與商務用 Skype 和較大型 Skype 網路整合即時共同作業模型的 web 體驗。

- **[語音服務 Api](/azure/cognitive-services/speech-service/)**<br>
辨識語音、合成語音、取得即時翻譯、轉譯交談，或將語音整合到您的 Bot 體驗中。

- **[文字分析 API](/azure/cognitive-services/text-analytics/)**<br>
提供對原始文字的 advanced 自然語言處理，並包含四個主要功能：情感分析、關鍵字組解壓縮、命名實體辨識和語言偵測。

- **[整合通訊 Web API 2。0](/skype-sdk/ucwa/unifiedcommunicationswebapi2_0)**<br>
    公開商務用 Skype Server 2015 立即訊息 (IM) 和狀態功能的 REST API。

- **[影片索引器](/azure/media-services/video-indexer/)**<br>
Azure 媒體播放機影片索引子可讓您使用影片與音訊模型，從影片中取出見解。 產生穩定的影片輸出、偵測動作、建立智慧縮圖，以及偵測和追蹤臉部。

- **[VS Code API](https://code.visualstudio.com/api/references/vscode-api)**<br>
您可以在 Visual Studio Code 擴充功能中叫用的一組 JavaScript Api。 [擴充功能 API](https://code.visualstudio.com/api)可讓您自訂和增強內建至 VS Code 的擴充功能。

- **[Yammer REST API](https://developer.yammer.com/docs/getting-started)**<br>
使用 Yammer 在您的組織中建立社區和協同作業。 此平臺是由一組開放 Api 和工具所組成，可讓您將 Yammer 的社交服務帶入您的應用程式。
