# <a name="accessibility"></a>協助工具選項  

協助工具是關於建立體驗, 讓您的通用 Windows 平臺 (UWP) 應用程式可供在各種環境中使用技術, 並以各種需求和體驗來處理 UI 的人員使用。 根據法律的規定，某些情況必須提供協助工具。 不過無論法律的規定為何，最好能解決無障礙問題，這樣應用程式便能吸引更多的使用者。

> 您的應用程式的協助工具也有 Microsoft Store 的聲明!
