# <a name="sessionssessionidscidsscid"></a>/sessions/{sessionId}/scids/{scid}
擷取此儲存體類型的配額資訊。 這些 Uri 的網域是`titlestorage.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| sessionId| 字串| 工作階段的識別碼來查閱。| 
| scid| guid| 要查閱的服務組態的識別碼。| 
  
<a id="ID4E3B"></a>

 
## <a name="valid-methods"></a>有效的方法

[GET](uri-sessionssessionidscidsscid-get.md)

&nbsp;&nbsp;擷取此儲存體類型的配額資訊。
