# <a name="supported-banner-ad-sizes"></a>支援的橫幅廣告大小

通用 Windows 平台 (UWP) 應用程式支援下列橫幅 ad 大小。 當您在您的應用程式產生**AdControl**物件時，請確定您設定以符合這些支援大小之一的高度和寬度屬性。

* 160x600
* 300x50
* 300x250
* 300x600
* 320x50
* 640x100
* 728x90

> [!NOTE]
> 最佳的體驗，我們建議在 Windows 10 行動裝置上使用大小： 300 x 50、 320 x 50 和 640 x 100。
