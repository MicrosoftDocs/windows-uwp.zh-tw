# <a name="symbologies"></a>碼制
本主題提供 Windows 10 隨附的軟體條碼解碼器所支援之每個條碼符號表示的範例條碼, 包括:UPC/EAN, 程式碼 39, 程式碼 128, 交錯2之 5, 資料條形全方向, 資料條形堆疊, QR 代碼和 GS1DWCode。
