# <a name="leaderboards-uris"></a>排行榜 URI

本節提供從 Xbox Live 服務的通用資源識別元 (URI) 位址和相關聯的超文字傳輸通訊協定 (HTTP) 方法的詳細*排行榜*。

這些 Uri 的網域是 leaderboards.xboxlive.com。

<a id="ID4EDB"></a>


## <a name="in-this-section"></a>本節內容

[/scids/{scid}/leaderboards/{leaderboardname}](uri-scidsscidleaderboardsleaderboardname.md)

&nbsp;&nbsp;存取預先定義的全球排行榜。

[/users/xuid({xuid})/scids/{scid}/stats/{statname)/people/{all\|favorite}](uri-usersxuidscidstatnamepeople.md)

&nbsp;&nbsp;存取社交 （等級） 排行榜。
