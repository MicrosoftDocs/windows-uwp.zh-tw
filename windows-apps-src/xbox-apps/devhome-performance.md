# <a name="performance-page-dev-home"></a>效能頁面 （開發人員首頁）
   
  
開發人員首頁的 [**效能**] 頁面上會模擬各種磁碟活動和為了疑難排解而 CPU 使用量條件。   
 ![效能] 頁面上的開發人員首頁](images/devhome_performance.png)   
  
[**效能**] 頁面包含下列窗格：   
 
   *  [磁碟模擬](#ID4EEB)  
   *  [CPU 模擬](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="disk-simulation"></a>磁碟模擬  
   
  
**磁碟模擬**窗格會模擬從系統或其他應用程式的磁碟活動。   
  
<a id="ID4EOB"></a>

   

## <a name="cpu-simulation"></a>CPU 模擬  
   
  
**CPU 模擬**窗格會模擬 CPU6 上的活動 （共用的核心）。   
  
<a id="ID4EYB"></a>
