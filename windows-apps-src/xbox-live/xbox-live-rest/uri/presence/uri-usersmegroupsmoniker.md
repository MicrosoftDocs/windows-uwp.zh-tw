# <a name="usersmegroupsmoniker"></a>/users/me/groups/{moniker}
存取 PresenceRecord 適合我的群組。 這些 Uri 的網域是`userpresence.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| moniker| 字串| 定義使用者群組的字串。 目前唯一接受的 moniker 是 「 人 」，以大寫 'P'。|
