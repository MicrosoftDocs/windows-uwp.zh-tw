# <a name="windows-ml"></a>Windows ML

Windows 機器學習 (ML) 可讓您在應用程式中使用訓練過的機器學習模型。 此平台在 Windows 10 裝置上本機評估訓練過的模型，透過利用裝置的 CPU 或 GPU提供硬體加速效能，並計算傳統 ML 演算法和深度學習的評估。
