# <a name="media-capture-page-dev-home"></a>媒體擷取頁面 （開發人員首頁）
   
  
開發人員常用的**媒體擷取**頁面擷取目前執行主控台上的標題的影片。   
 ![擷取開發人員首頁] 的頁面](images/devhome_capture.png)   
  
**媒體擷取**頁面包含下列窗格：   
 
   *  [視訊擷取](#ID4EHB)  

 
<a id="ID4EHB"></a>

   

## <a name="video-capture"></a>視訊擷取  
   
  
**視訊擷取**窗格包含記錄從開發人員主控台的視訊資料流的工具。 您也可以擷取從主控台的最後一個幾秒的影片。
