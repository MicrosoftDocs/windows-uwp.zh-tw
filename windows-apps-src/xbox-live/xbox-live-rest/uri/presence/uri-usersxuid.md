# <a name="usersxuidxuid"></a>/users/xuid({xuid})
存取另一個使用者或用戶端的目前狀態。 這些 Uri 的網域是`userpresence.xboxlive.com`。
 
  * [URI 參數](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| xuid| 64 位元不帶正負號的整數| Xbox 使用者識別碼 (XUID) 的目標使用者。| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>有效的方法

[GET (/users/xuid({xuid}))](uri-usersxuidget.md)

&nbsp;&nbsp;探索其他使用者或用戶端的目前狀態。
